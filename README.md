# Droplet Tracker

Обработка видео с каплями воды (распознавание, отслеживание, определение скорости и угла движения).

## Структура проекта

Приложение разделено на три модуля:

- **`cli.py`** – обработка аргументов командной строки.
- **`process_data.py`** – функции для обработки одного кадра (`process_frame`) и сравнения капель между кадрами (`is_same_droplet`).
- **`main.py`** – главный модуль: функция `process_video()` выполняет цикл чтения видео, отслеживания и записи результатов, а `main()` связывает `cli` с `process_video`.

Точка входа: `main.py ...`.

## Настройка, сборка и запуск

### Настройка путей в `.env`

```env
INPUT_DIR=/путь/к/папке/с/видео
OUTPUT_DIR=/путь/к/папке/для/результатов
```

### Сборка и запуск

```bash
docker compose build
docker compose run --rm tracker input/input_video.mp4 output/output_video.avi output/data.csv
```

## Параметры

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|-----------|
| input_video | обязательный | Путь к исходному видео |
| output_video | обязательный | Путь для обработанного видео |
| data_csv | обязательный | Путь для CSV с результатами |
| `--position-threshold` | 100 | Максимальное расстояние между центрами капель (пиксели) |
| `--area-ratio` | 0.10 | Допустимое изменение площади (0.10 = 10%) |
| `--dim-ratio` | 0.10 | Допустимое изменение размеров (0.10 = 10%) |
